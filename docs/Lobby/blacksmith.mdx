# Schmiede
In der Schmiede kann man bei Thor die gesammelten Schmiedevorlagen (Armortrims) auf seine Rüstung anwenden. Zum Ändern des Rüstungsstils wird lediglich das gewünschte Rüstungsteil mit einem Rüstungsbesatz kombiniert. Eine Farbänderung durch Werkstoffe ist dabei nicht möglich.

import Admonition from '@theme/Admonition';

<Admonition type="tip" title="Tipp!">
Die Schmiedevorlage für die Netheritaufwertung kann zusätzlich an jedem beliebigen Schmiedetisch angewandt werden.
</Admonition>

Zudem ist es möglich, Custom Rüstungen bei Thor zu reparieren.
