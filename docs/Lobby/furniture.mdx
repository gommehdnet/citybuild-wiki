# Möbelhaus
Täglich ab 0 Uhr können im Möbelhaus neue Möbel gekauft werden, dabei braucht es neben Geld auch "Möbel-Tickets", welche durch Crates oder durch das bezwingen von Bossen erhalten werden kann.
Diese Möbel bieten meist einen rein dekorativen Zweck, auf manche kann man sich sogar setzen.
