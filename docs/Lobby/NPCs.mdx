# NPCs
In der Lobby gibt es eine Vielzahl von NPCs, welche einem Inhalte der Lobby näher bringen und beschreiben. 
So gibt es z.B. Hugo, welcher einem den Mülleimer erklärt, in welchem man Items reinlegen kann, die dann jeder Spieler rausnehmen kann. 
Tresorwächter: Dieser steht vor dem Tresor und weißt auf die Endertruhen und ihre Funktion hin.
Mob-Collection: Hierrüber kann man seine Mob-Collection und die bisher gesammelten Köpfe aufrufen, alternativ kann man auf den Befehl "/collection" nutzen 
Custom Craftingrezepte: Hier kann man sich die ganzen Rezepte zu Items ansachauen, die es extra in CityBuild gibt, alternativ kann man auch "/ci" nutzen.
Stadtwache: Diese hält einen immer auf dem laufenden, wenn man wissen möchte wo man Spieler oder Bugs melden kann
Mall: Hier lässt sich eine Zusammenfassung der Mall einsehen und wie man dort mit welchem Items Sachen kaufen kann.
Belohnungen: Hier können einmal täglich/wöchtentlich/monatlich gratis Items abgeholt werden.
CityBuild Gold-Shop: Hier können die Serverwährung Gold, Crates, Ränge und Universal-Booster gekauft werden.
Thor: Bei ihm kann man Schmiedevorlagen (Armortrims) auf seine Rüstung packen und seine Custom-Rüstung reparieren
