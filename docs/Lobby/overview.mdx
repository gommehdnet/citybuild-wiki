# Lobby - Übersicht
Die Lobby fungiert als Knotenpunkt für Farm-/Plotwelt, in welcher Events stattfinden können und in welcher man Items an den Server ver-/kaufen kann.
Weiterhin befindet sich im Spawnhaus ein Tresor, in welchem, abhängig vom CityBuild-Level, Endertruhen gekauft werden können, auf die man in ganz CityBuild über "/ec 1-12" zugreifen kann. Sechs Endertruhen können dabei normal erspielt werden und für 6 weitere braucht man Rechte-Bücher, welche es in der Crate gibt.
