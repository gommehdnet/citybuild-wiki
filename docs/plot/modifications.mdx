# Modifizierungen
Durch folgende Premium-/Crate-Rechte kann das Plot wie folgt geändert werden: 
/rand
Gibt dir eine Vielzahl an Blöcke zur Auswahl, durch welche du den Rand deines Plots ändern kannst durch "/rand reset" kannst du ihn auf den Standard zurück stellen
/wand
Gibt dir eine Gibt dir eine Vielzahl an Blöcke zur Auswahl, durch welche du die Wand deineds PLots ändern kannst, durch "/wand reset" kannst du ihn auf den Standard zurück stellen
/plot biom
Durch "/plot biom" kannst du das Biom deines Plots frei wählen, dabei gibt es die Möglichkeit es für das ganze Plot zu ändern (/plot biome set biom), nur für einen Teil des Plots zu ändern (/plot biome setsubplot Biom) oder es auf einem ganz bestimmten Bereich zu ändern
