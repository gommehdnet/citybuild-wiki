# Plot Config
Über "/plot config" kann man auf seinem Plot ein Menü aufrufen, in welchem man eine Vielfalt von Einstellungen für das Plot anpassen kann. Einige Einstellungen lassen sich aber nur mithilfe eines Ranges einstellen.
So lässt sich z.B. einstellen, dass jede auf Werkbänke oder Zaubertische zugreifen kann oder sogar das niemand Items aufsammeln kann.
